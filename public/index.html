Como funciona
1
Exporta a contabilidade
O sistema lê a coluna Descrição e extrai os cheques de linhas que contenham a palavra CHEQUE. Formato esperado: CHEQUE 164709411
2
Importa os extratos bancários
O match é feito pelo número do cheque — o mesmo que aparece nos dois ficheiros.
CGD
CAIXA BR10164709411 164709411
→ extrai último número
Novo Banco
CHEQUE 164709411
→ número após CHEQUE
BCP
CHEQUE 164709411
→ número após CHEQUE
3
Reconciliação automática
Cheques encontrados no extrato bancário → Reconciliados com data de desconto. Os restantes ficam Pendentes com dias em aberto e alertas a partir dos 30 dias.
